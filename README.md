## Método 50/30/20 — Organizador Financeiro
Aplicação web interativa para organizar o salário seguindo o método 50/30/20: 50% para necessidades, 30% para desejos pessoais e 20% para poupança.
Desenvolvido com Python e Streamlit como projeto de aprendizado de programação.

## Funcionalidades

Calculadora automática — informe sua renda líquida e veja a divisão nas três categorias em reais
Registro de gastos — adicione despesas com descrição, valor e categoria
Barras de progresso — acompanhe quanto já foi gasto em relação ao limite de cada categoria
Gráfico de pizza — visualize a distribuição real dos seus gastos
Persistência local — os dados são salvos em arquivo JSON entre as sessões


## Tecnologias

Python 3
Streamlit — interface web interativa
Plotly — gráficos interativos
Pandas — manipulação de dados

## Como rodar localmente
1. Clone o repositório
bashgit clone https://github.com/andressamarques-s/Projeto-Controle-Financeiro.git
cd seu-repositorio
2. Instale as dependências
bashpip install -r requirements.txt
3. Execute o app
bashstreamlit run app.py
O app abrirá automaticamente no navegador em http://localhost:8501.
